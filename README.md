# zomato-restaurant-analysis
Zomato Restaurant Analysis

Project Overview

This project analyzes restaurant data from Zomato Bangalore to understand customer preferences, restaurant distribution, ratings, pricing, cuisines, and online ordering trends.

The goal was to perform Exploratory Data Analysis (EDA), clean the dataset, visualize important patterns, and generate business insights that could help restaurant owners and food delivery platforms make data-driven decisions.

⸻

Dataset Information

* Source: Kaggle Zomato Bangalore Restaurants Dataset
* link: https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants
* Total Records: 51,717
* Total Features: 17

Key Features:

* Restaurant Name
* Location
* Rating
* Votes
* Online Order
* Table Booking
* Cuisine
* Restaurant Type
* Approximate Cost for Two

Project Workflow

1. Data Collection

Loaded the Zomato Bangalore restaurant dataset from CSV format.

2. Data Understanding

* Examined dataset structure
* Reviewed columns and data types
* Identified missing values

3. Data Cleaning

* Removed duplicate records
* Handled missing values
* Cleaned rating column
* Converted rating values to numeric format
* Cleaned and converted cost column

4. Exploratory Data Analysis (EDA)

Performed analysis on:

* Restaurant ratings
* Restaurant locations
* Online ordering trends
* Table booking trends
* Popular cuisines
* Restaurant types
* Customer votes
* Cost distribution

5. Data Visualization

Created charts to identify trends and patterns in the dataset.

Key Business Insights

Restaurant Locations

Certain locations contain significantly higher numbers of restaurants, indicating strong demand and high competition.

Online Ordering

A large percentage of restaurants support online ordering, highlighting the importance of delivery services.

Customer Ratings

Most restaurants have ratings between 3.5 and 4.2, suggesting generally positive customer experiences.

Popular Restaurant Types

Casual Dining and Quick Bites are among the most common restaurant categories.

Popular Cuisines

North Indian, South Indian, Chinese, and Biryani-based cuisines dominate customer preferences.

Customer Engagement

Restaurants with higher ratings generally receive more votes and customer engagement.
